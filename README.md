# Retails-Mart-SQL-Project
Retail Store SQL Project using MySQL Workbench. Includes joins, subqueries, views, stored procedures, triggers, and business insights.

## Objective
Analyze sales, customers, products and orders using SQL.

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## Features
- Joins
- Subqueries
- Views
- Stored Procedures
- Triggers
- Aggregations

## Database Tables
- customers
- products
- orders
- order_items

## Sample Insights
- Top selling products
- Revenue by category
- Customer purchase history
- Order status analysis
